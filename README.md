# Mini Projects — Java Application Collection

This repository contains a collection of small Java applications developed to explore core programming concepts such as Object-Oriented Programming, file handling, console interaction, and basic GUI development using Java Swing.

Each project is self-contained and focuses on solving a specific problem or simulating a real-world system.

---

## Projects Included

### Railway Reservation System

A desktop-based railway ticket booking application built using Java Swing.  
The system allows users to book tickets, generate PNR numbers, view reservations, and cancel bookings while maintaining persistent ticket storage.

Core concepts demonstrated:
- Java Swing GUI development
- Modular application architecture
- File-based reservation storage
- Event-driven UI design

---

### Online Examination System

A Java Swing application that simulates a digital exam environment.  
Users can log in, attempt multiple-choice questions, track time during exams, and view results along with leaderboard rankings.

Core concepts demonstrated:
- GUI application development
- Authentication and user management
- Timed assessments
- Persistent leaderboard storage

---

### ATM Interface Simulation

A console-based ATM simulation that allows users to perform basic banking operations such as checking account balance, depositing funds, withdrawing money, and viewing transaction history.

Core concepts demonstrated:
- Object-Oriented Programming
- Transaction management
- Console-based interaction
- Persistent account data handling

---

### Number Guessing Game

A console-based number guessing game where players attempt to guess a randomly generated number within a specific range.

Core concepts demonstrated:
- Game logic implementation
- Random number generation
- Score tracking and leaderboard persistence
- Console UI interaction
